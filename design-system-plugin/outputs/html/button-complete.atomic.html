Docs Contract

Button component is used to trigger an action or event, such as submitting a form, opening a Dialog, canceling an action, or performing a delete operation.

Prop Required Default Description
variant no ButtonVariant.PRIMARY Visual style variant of the button.
size no ButtonSize.MEDIUM Size of the button.
label yes none Text content displayed inside the button. Required for the accessible name.
icon no none Icon to display alongside the label.
iconPosition no IconPosition.START Position of the icon relative to the label.
inverse no false Applies inverse styling for use on dark backgrounds.
fullWidth no false Expands the button to fill its container width.
disabled no none Disables the button.
isLoading no false Displays a loading spinner and disables interactions while an action is in progress.
as no none Overrides the default HTML tag. Use anchor semantics only for navigation/link behavior.
href no none URL used when the button is rendered as an anchor.
target no none Target window for anchor usage.
brandTheme no BrandTheme.primary Overrides the theme at component level.
