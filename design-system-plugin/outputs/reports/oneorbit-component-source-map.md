该报告对应两份 HTML 输出：复杂查询列表和消息待办首页。分类含义：`runtime component` 表示走本地 Sanofi runtime；`docs-backed primitive` 表示本地有组件文档和 primitive contract，但未纳入首轮 runtime；`explicit fallback module` 或 `compound fallback pattern` 表示本地缺少对应 runtime primitive，已显式声明原因。
