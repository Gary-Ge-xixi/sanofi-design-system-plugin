# Sanofi Elements Local Registry

This folder is generated by `scripts/build-local-registry`.

Use this registry as the first lookup layer for AI prototype generation:

1. Search `registry.json` or run `scripts/search-local-registry "<prompt>"`.
2. Load the matched `items/*.json` record.
3. Load only the files declared by that item.
4. Run resolver, renderer, and validators.

Normal generation should not call remote Storybook or Zeroheight pages. When upstream sources change, refresh the local Markdown, contracts, and tokens first, then rebuild the registry.
