# Radius

The radius tokens are used to define the border radius of the components. The naming
convention follows the t-shirt sizing system too.

### CSS Variables

```
--elements-radius-none: 0px;
--elements-radius-sm: 4px;
--elements-radius-md: 8px;
--elements-radius-lg: 12px;
--elements-radius-xl: 16px;
--elements-radius-rounded: 9999px;
```
